# Simple Chat Application

    This project is about chat application and the purpose of this project is to learn about

- websocket
- nodejs express
- redis

## Command for run

###### - run nodejs on port localhost 3000 
###### - redis on port 6379

```bash
$ docker-compose up -d
```
